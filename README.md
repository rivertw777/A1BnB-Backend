# 🖥 A1BnB 프로젝트

### 소개
A1BnB는 딥러닝 Object Detection, Classification을 활용하여, 숙박공유 사이트 호스트가 사진을 업로드해서 바로 어떤 공간인지, 어떤 어메니티가 있는지 작성해주는 서비스 입니다. 
## 팀프로젝트 A1BnB를 진행하며 아쉬웠던 점을 개인적으로 새로운 기술 스택 및 기능 구현을 추가하여 재구현하였습니다.
